== Description ==

By using this smart plugin, allows you to Hide product content from  wooocommerce Single product page, Shop page and category page , no need to edit theme and woocommerce plugin!.

This Plugin integrates seamlessly with any WordPress Theme, and Admin can easily remove different content of product like price, thumbnail, product tab, add to cart button etc.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

After active plugin go to Woocommerce->Adding Content to the Footer.
